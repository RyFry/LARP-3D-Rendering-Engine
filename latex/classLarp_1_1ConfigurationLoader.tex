\subsection{Detailed Description}
Used to load basic configurations such as screen width and height so that recompiling for simple changes is not necessary.

\begin{DoxyNote}{Note}
If options are provided that are not part of the pre-\/defined acceptable options, an error message is printed to the console, but execution will not be stopped. 

If pre-\/defiined options are not given, then the default settings defined in \hyperlink{classLarp_1_1ConfigurationLoader_a5b5d1d47f1f5cced025a85d248ae57e7}{Configuration\-Loader\-::\-D\-E\-F\-A\-U\-L\-T\-S} for the missing options will be used. 

If the given file cannot be opened, then all configurations defined in \hyperlink{classLarp_1_1ConfigurationLoader_a5b5d1d47f1f5cced025a85d248ae57e7}{Configuration\-Loader\-::\-D\-E\-F\-A\-U\-L\-T\-S} will be used.
