glm::vec3 Larp::DirectionalLight::_ambient

Intensity of this DirectionalLight's ambient light. Ambient light is usually set to a low intensity because the ambient color should not be too dominant.

glm::vec3 Larp::DirectionalLight::_diffuse

Intensity of this DirectionalLight's diffuse light. Diffuse light is usually set to to the exact color the user would like the light to have; often a bright white color.

glm::vec3 Larp::DirectionalLight::_direction

The direction the DirectionalLight is aiming

glm::vec3 Larp::DirectionalLight::_specular

Intensity of this DirectionalLight's specular light. Specular light is usually kept at glm::vec3(1.0f) shining at full intensity.
