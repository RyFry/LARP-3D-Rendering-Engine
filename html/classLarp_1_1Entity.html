Draws the model atteched to this entity using the associated shader

Parameters
modelThe parent's model matrix. Used to calculate this Entity's model matrix prior to rendering
viewThe view matrix to apply during rendering. This should be obtained from a Camera object.
projectionThe projection matrix to apply during rendering. This should also be obtained from a Camera object.
view_posThe camera view position to apply during rendering. This should be obtained from a Camera object and passed in via the SceneGraph.
directional_lightsThe vector of directional lights currently held by the SceneGraph. If the vector is not empty, these should be incorporated in the lighting calculations.
point_lightsThe vector of point lights currently held by the SceneGraph. If the vector is not empty, these should be incorporated in the lighting calculations.
spot_lightsThe vector of spot lights currently held by the SceneGraph. If the vector is not empty, these should be incorporated in the lighting calculations.
Warning
This method assumes that each model has provided shaders with variables matching those specified in the method.
