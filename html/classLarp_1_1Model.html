Loads the textures for the material. After this function runs, this Model's _loaded_textures will be populated with all of the Textures necessary to render the Model.

Parameters
matThe material to load textures for.
typeThe type of the texture to load. Should be one of aiTextureType_DIFFUSE, or aiTextureType_SPECULAR.
texture_typeThe type of the generated Texture object. Should be one of Texture::DIFFUSE or Texture::SPECULAR.
Returns
A vector of loaded Textures
Note
type and texture_type should both be the respective type (i.e. DIFFUSE or SPECULAR).
