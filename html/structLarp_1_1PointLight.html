Sets the linear and quadratic factors used in this PointLight's attenutation calculation.

Parameters
linearThe new linear factor of this light.
quadraticThe new quadratic factor of this light.
Note
Attenutation reduces the intensity of a light, over the distance a light ray travels. Setting the right values depends on many factors such as the environment, distance the light should cover, and they type of light. Below there are some good starting points for most lights:

| Distance | Linear | Quadratic |

| 7 | 0.7 | 1.8 |

| 13 | 0.35 | 0.44 |

| 20 | 0.22 | 0.20 |

| 32 | 0.14 | 0.07 |

| 50 | 0.09 | 0.032 |

| 65 | 0.07 | 0.017 |

| 100 | 0.045 | 0.0075 |

| 160 | 0.027 | 0.0028 |

| 200 | 0.022 | 0.0019 |

| 325 | 0.014 | 0.0007 |

| 600 | 0.007 | 0.0002 |

| 3250 | 0.0014 | 0.000007 |

Member Data Documentation

glm::vec3 Larp::PointLight::_ambient

Intensity of this PointLight's ambient light. Ambient light is usually set to a low intensity because the ambient color should not be too dominant.

const GLfloat Larp::PointLight::_constant = 1.0f
static

A constant value used to reduce the PointLight's intensity over a distance. This is usually kept at 1.0 to make sure the denominator of the equation this value is used in is never smaller than 1.

glm::vec3 Larp::PointLight::_diffuse

Intensity of this PointLight's diffuse light. Diffuse light is usually set to to the exact color the user would like the light to have; often a bright white color.

GLfloat Larp::PointLight::_linear

This linear term is used as part of the attenuation equation which reduces a light's intensity of a distance.

glm::vec3 Larp::PointLight::_position

This PointLights's position

GLfloat Larp::PointLight::_quadratic

This quadratic term is used as part of the attenuation equation which reduces a light's intensity of a distance.

glm::vec3 Larp::PointLight::_specular

Intensity of this PointLight's specular light. Specular light is usually kept at glm::vec3(1.0f) shining at full intensity.
