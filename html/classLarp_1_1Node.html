Draws the Entity attached to this node using this Node's derived model matrix, then recursively calls the draw method on each of this Node's children.

Parameters
view_posThe camera view position to apply during rendering. This should be obtained from a Camera object and passed in via the SceneGraph.
directional_lightsThe vector of directional lights currently held by the SceneGraph.
point_lightsThe vector of point lights currently held by the SceneGraph.
spot_lightsThe vector of spot lights currently held by the SceneGraph.
Note
the order in which the draw method is called on the Node's children is not necessarily the order in which they were created.
